JavaScript Events

JavaScript Events are actions performed by the user or the browser. JavaScript reacts to these actions using event handlers.

Types of JavaScript Events

JavaScript events are broadly classified into the following types:

  • Mouse Events
  • Keyboard Events
  • Form Events
  • Window / Document Events
  • Clipboard Events

1. Mouse Events

Mouse events occur when the user interacts with the mouse.

  • click
  • dblclick
  • mouseover
  • mouseout
  • mousedown
  • mouseup
Hover or click here

2. Keyboard Events

Keyboard events occur when the user presses keys on the keyboard.

  • keydown
  • keyup
  • keypress (deprecated)

3. Form Events

Form events occur when the user interacts with form elements.

  • submit
  • change
  • focus
  • blur
  • reset




4. Window / Document Events

These events are related to the browser window or document.

  • load
  • resize
  • scroll
  • unload
Resize the browser window to see the effect

5. Clipboard Events

Clipboard events occur when the user copies, cuts, or pastes content.

  • copy
  • cut
  • paste
