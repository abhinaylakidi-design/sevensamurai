DOM (Document Object Model) in JavaScript

1. What is DOM?

The DOM (Document Object Model) is a programming interface that represents an HTML document as a tree of objects. JavaScript uses the DOM to access, modify, add, or remove HTML elements dynamically.

<html>
  <body>
    <h1>Hello DOM</h1>
    <p>This is a paragraph</p>
  </body>
</html>

In the DOM tree:
• document is the root
• html is the parent node
• h1 and p are child nodes

2. Accessing DOM Elements

JavaScript provides different methods to select HTML elements.

document.getElementById("id")
document.getElementsByClassName("class")
document.getElementsByTagName("tag")
document.querySelector("css-selector")
document.querySelectorAll("css-selector")

3. Modifying DOM Content

Once an element is selected, we can change its content using:

element.innerHTML = "New HTML Content";
element.innerText = "New Text Content";

4. Modifying Styles Using DOM

JavaScript can dynamically change CSS styles using the style property.

Click the button to change my style

5. Working with Attributes

Attributes like src, href, class can be accessed or changed using DOM.

element.setAttribute("attribute", "value");
element.getAttribute("attribute");
element.removeAttribute("attribute");

6. DOM Events

Events allow JavaScript to respond to user actions such as clicks, typing, or mouse movements.
