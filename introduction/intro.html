JavaScript – Detailed Explanation with Examples

1. What is JavaScript?

JavaScript is a scripting language used to make web pages interactive and dynamic. It runs inside the browser.

  • Adds interactivity to web pages
  • Used for validation, animations, events
  • Works with HTML and CSS
✔ HTML = Structure
✔ CSS = Styling
✔ JavaScript = Behavior

2. Ways to Add JavaScript

a) Inline JavaScript

<button onclick="alert('Hello!')">Click Me</button>

b) Internal JavaScript

<script>
  alert("Hello World");
</script>

c) External JavaScript

<script src="script.js"></script>

3. Variables in JavaScript

Variables store data values.

let name = "Abhinay";
const age = 21;
var city = "Hyderabad";
  • let – block scoped
  • const – cannot be changed
  • var – function scoped (old)

4. Data Types

let number = 10;        // Number
let text = "Hello";    // String
let status = true;     // Boolean
let value = null;      // Null
let data;              // Undefined

5. Functions in JavaScript

A function is a reusable block of code that performs a task. 

function greet() {
  alert("Welcome to JavaScript");
}

6. Events in JavaScript

Events occur when the user interacts with the page. 

<button onclick="showMessage()">Click</button>

<script>
function showMessage() {
  alert("Button Clicked!");
}
</script>

7. DOM (Document Object Model)

DOM allows JavaScript to access and modify HTML elements. 

document.getElementById("demo").innerHTML = "Text Changed!";

8. Conditional Statements

let age = 18;

if (age >= 18) {
  console.log("Eligible to vote");
} else {
  console.log("Not eligible");
}

9. Loops in JavaScript

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

10. Form Validation (Basic)

function validate() {
  let name = document.getElementById("name").value;

  if (name === "") {
    alert("Name is required");
    return false;
  }
  return true;
}
