Regular Expressions (RegEx) in JavaScript

1. What is a Regular Expression?

A Regular Expression (RegEx) is a pattern used to search, match, or replace text.

It is commonly used for:

  • Validating emails and phone numbers
  • Searching text
  • Replacing words

2. How to Create a RegEx

There are two ways to create a RegEx:

// Method 1: Using slashes
let pattern1 = /hello/;

// Method 2: Using RegExp constructor
let pattern2 = new RegExp("hello");

3. Common RegEx Methods

test() – Checks if pattern exists

let text = "Hello World";
let pattern = /World/;

let result = pattern.test(text);
console.log(result); // true
Output: true (because "World" exists)

match() – Finds matching values

let text = "I love JavaScript";
let result = text.match(/JavaScript/);
console.log(result);
Output: ["JavaScript"]

replace() – Replaces matched text

let text = "I like cats";
let result = text.replace(/cats/, "dogs");
console.log(result);
Output: I like dogs

4. Important RegEx Symbols

.   → Any single character
\d  → Any digit (0–9)
\w  → Any word character (a-z, A-Z, 0-9)
\s  → Space
^   → Starts with
$   → Ends with
+   → One or more times
*   → Zero or more times

5. Example: Check Mobile Number

Mobile number should contain exactly 10 digits:

let mobile = "9876543210";
let pattern = /^\d{10}$/;

console.log(pattern.test(mobile));
Output: true

6. Example: Check Email

let email = "test@gmail.com";
let pattern = /^[a-zA-Z0-9._]+@[a-zA-Z]+\.[a-zA-Z]{2,}$/;

console.log(pattern.test(email));
Output: true

7. Flags in RegEx

i → Case insensitive
g → Global search
m → Multiline
  
let text = "Hello hello";
let pattern = /hello/gi;

console.log(text.match(pattern));
Output: ["Hello", "hello"]

8. Summary

✔ RegEx is used for pattern matching
✔ Helps in validation and text manipulation
✔ Very powerful and commonly used in JavaScript
