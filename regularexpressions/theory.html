Regular Expressions (RegEx) in JavaScript

1. What is a Regular Expression?

A Regular Expression (RegEx) is a pattern used to match, search, validate, or replace text. In JavaScript, RegEx is commonly used for form validation, searching strings, and data filtering.

let pattern = /hello/;

2. Creating a Regular Expression

There are two ways to create a RegEx in JavaScript:

/pattern/flags new RegExp("pattern", "flags")

3. test() Method

The test() method checks whether a pattern exists in a string. It returns true or false.

4. match() Method

The match() method returns the matched value or null if no match is found.

let text = "JavaScript is powerful"; text.match(/script/i);

5. Character Classes

Character classes match specific types of characters.

\d → digits (0-9)
\w → letters, numbers, underscore
\s → whitespace
. → any character

7. RegEx Flags

Flags modify how a pattern works.

i → case-insensitive
g → global search
m → multi-line

8. replace() Method

The replace() method replaces matched text.

let text = "I love JavaScript"; text.replace(/JavaScript/, "JS");
